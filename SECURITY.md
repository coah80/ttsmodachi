# Security

please do not open issues with real tokens, ROMs, server IPs, private logs, or personal Discord data.

if you accidentally publish a Discord bot token:

1. reset the token in the Discord Developer Portal
2. update `.env`
3. restart the bot
4. remove the token from git history if it was committed

this repo is meant to be self-hosted. the renderer should stay private unless you have a reverse proxy and token setup around it.
